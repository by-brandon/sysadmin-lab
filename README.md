# sysadmin-lab
### Windows Server & Infrastructure Lab

This repository documents my personal homelab used to practice system administration, Windows Server, and infrastructure fundamentals.  
The lab is designed to simulate a small enterprise environment and to build hands-on experience with System Administration tasks.

---

## Goals

- Develop practical Windows Server administration skills
- Gain hands-on experience with Active Directory and core network services
- Practice troubleshooting, documentation, and change tracking

---

## Lab Overview

- **Hypervisor:** Proxmox
- **Server OS:** Windows Server (2025, might do 22 and try upgrading to 25)
- **Client OS:** Windows 10/11
- **Network:** Isolated internal network (pfSense / NAT / internal switch)

---

## Planned Services

- Active Directory Domain Services (AD DS)
- DNS
- DHCP
- Group Policy
- File services with NTFS permissions
- Basic monitoring and logging

Services will be documented individually as they are implemented.
